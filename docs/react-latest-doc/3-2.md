# 几个大坑的解读



## 坑1：Hooks+控制流

```tsx
if(...) {
  useEffect(() => {
    
  }, [])   
}
```



## 坑2 ： Stackoverflow



```tsx
const [s, setS] = useState(0)

useEffect(() => {
   setS(x => x + 1)
}, [s])
```



## 坑3： Effect的注销

```tsx
useEffect(() => {
   editor.on("some-event" ,() => {
       
   })
}, [])
```



## 坑4： 多状态更新



```tsx
const [money, setMoney] = useState(...)
const [unit, setUnit] = useState(...)
                                 
// some click function
setMoney(100)
setUnit("$")
```
